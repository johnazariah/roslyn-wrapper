namespace (...)
Signature: namespaceName:string -> {:'?7673 -> usings:string list -> members:seq<(type)> -> }:'?7674 -> (type)
Type parameters: '?7673, '?7674

This function creates a 'namespace' with the given name and contents:

Parameters
  1. namespaceName : string : The name of the namespace to be created
  2. { : white noise : ignored - use the { constant for visual structure
  3. usings : string seq : The namespaces to reference within this namespace
  4. members : MemberDeclaration seq : the members of this namespace. Typically class and interface
  5. } : white noise : ignored - use the } constant for visual structure
Returns

A namespace object
